# AI-Virtual-Mouse-Air-Writing

An AI-powered virtual mouse and air writing system using hand gesture recognition with OpenCV and MediaPipe.

## 🚀 Features
* **Virtual Mouse:** Control your mouse cursor and clicks using hand gestures.
* **Air Writing:** Draw and write on a virtual canvas using finger movements.
* **Hand Tracking:** Accurate real-time landmark detection using MediaPipe.

## 🛠️ Tech Stack
* Python
* OpenCV
* MediaPipe
